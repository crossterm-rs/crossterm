# Crossterm Examples

The examples are compatible with the latest release.  

## Structure

```
├── examples
│   └── interactive-test
│   └── event-*
│   └── stderr
```

* `examples/interactive-test`, interactive demo
* `event-*`, event reading demo
* `stderr` , crossterm over stderr demo

## Run examples

```bash
$ cargo run --example [file name]
```
